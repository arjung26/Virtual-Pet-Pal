# Virtual Pet Pal

A desktop virtual pet application built with Python and Tkinter.

## Features
- Selectable pets with animations
- Health and mood management
- Play, feed, and sleep behaviors
- Transparent floating window UI

## How to Run
Run the `main.py` script to start the application.

## Requirements
- Python 3.x
- Pillow (`pip install pillow`)
